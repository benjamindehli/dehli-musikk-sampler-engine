# Code of Conduct

This project should be a friendly place. Everyone participating in it, in issues, pull requests, discussions and any other project space, is expected to act accordingly.

## Expectations

* Be respectful and constructive. Disagree with ideas, not with people.
* Assume good faith and be patient, especially with newcomers and with people writing in a second language.
* Harassment, personal attacks, discriminatory remarks and trolling are not tolerated.
* Keep discussions on topic and keep private information private.

## Enforcement

Violations can be reported to the maintainer, Benjamin Dehli, by opening an issue or, when the matter is sensitive, through GitHub's private vulnerability reporting form on this repository with a note that it concerns conduct. Reports are handled confidentially.

The maintainer may edit or remove comments, commits, issues and other contributions that violate this code of conduct, and may temporarily or permanently ban contributors for behavior deemed inappropriate, threatening or harmful.

This code of conduct is inspired by the [Contributor Covenant](https://www.contributor-covenant.org).
